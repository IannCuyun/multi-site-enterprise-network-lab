# Multi-Site Enterprise Network Lab

## Overview
This project is a beginner-to-intermediate virtual enterprise network lab built to simulate a small multi-site business environment.

The goal is to learn and demonstrate:
- network segmentation
- routing and firewalling
- Windows Server administration
- Active Directory and DNS
- Linux server administration
- client/server connectivity
- documentation and troubleshooting

## Status
Project setup in progress.

## Planned Technologies
- VirtualBox
- pfSense
- Windows Server
- Ubuntu Server
- Windows client VM
